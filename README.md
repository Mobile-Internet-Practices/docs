## 程序功能

### 概述

- **用户可以分享自己和宠物的日常生活**                   

- **用户可以为身边的流浪动物发布领养信息**

- **用户可以从领养信息中选择想要领养的宠物**     

- **用户可以关注自己感兴趣的博主及其宠物**             

- **用户可以查看收藏的宠物动态**                

- **用户可以在动态详情页对动态进行评论**                                

- **用户可以查看自己的浏览历史**              

- **用户可以查看自己发布的动态**           

- **用户可以上传宠物视频和照片**

- **用户可以浏览照顾宠物的注意事项和教程等** 

- **用户可以发布宠物相关的问题**

- **用户可以回答他人提出的问题**
